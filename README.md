# Zec-timing
Repository for **Z**ero**E**mission**C**hallenge-Timing by Darnhofer and Maderbacher
- ZEC-API branch by Darnhofer

## Services Overview
| Service             | Description |
|---------------------|-------------|
| User service        | Manages user accounts, profiles, and basic user-related data. |
| Attempt service     | Handles teams attempts for challenges. |
| Score service       | Calculates and stores scores/penalties for attempts.|
| Challenge service   | Provides challenge definitions and parameters. |
| Team/Driver service | Manages teams, drivers, and their relationships. |
| Auth service        | Handles authentication, authorization, and token management. |
